# AI4SecureCI

AI4SecureCI is an educational and research project on AI/ML-assisted malware detection and malware-family classification in cloud cyberinfrastructure.

## Website

Public website:

https://ai4secureci.github.io/ai4secureci/

## Repository

https://github.com/ai4secureci/ai4secureci

## Project Focus

- Online malware detection in cloud CI
- Runtime behavior monitoring
- Static and dynamic malware-analysis concepts
- Machine-learning-based malware detection
- Malware-family classification using behavioral and API-call data
- LLM-assisted cybersecurity education
- Responsible, isolated, and ethical lab design

## Curriculum Sections

- Module 3: Online Malware Detection in Cloud CI
- Lab 3: Online Malware Detection
- Module 4: Classifying Malware Threats in Cloud CI
- Lab 4: Malware Family Classification

## Safety Notice

This repository is for educational documentation, sanitized examples, research summaries, and safe instructional materials only.

Do not upload:

- Malware binaries
- Live malware samples
- Credentials or API keys
- Raw sensitive logs
- Cloud account secrets
- Unsafe execution instructions
- VM images containing malware
