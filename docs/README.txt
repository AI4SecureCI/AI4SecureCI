AI4SecureCI website
===================

This repository contains the public website for the AI4SecureCI curriculum.

Pages
-----
- index.html
- modules.html
- award_details.html
- contact.html

Overview
--------
AI4SecureCI is a curriculum-oriented project for AI/ML-assisted cloud cyberinfrastructure security education. The website presents curriculum scope, modules, lab structure, safety practices, and contact information.

The current pages emphasize:
- Cloud cyberinfrastructure security education.
- Module 3: Online Malware Detection in Cloud CI.
- Lab 3: safe behavior collection and ML-based online detection.
- Module 4: Malware Threat and Family Classification in Cloud CI.
- Lab 4: API-call behavioral datasets, family classification, model evaluation, and LLM-assisted interpretation.
- Defensive lab design with isolated VMs, clean ML analysis environments, and cloud CI monitoring demonstrations.

Publishing
----------
For GitHub Pages, place the website files in the selected publishing folder, such as docs/.

Recommended layout:

docs/index.html
docs/modules.html
docs/award_details.html
docs/contact.html
docs/README.txt

Safety note
-----------
This website is for cybersecurity education and defensive analysis. Malware-related content should be handled only in controlled, isolated, and policy-compliant lab environments.
