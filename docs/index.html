Cloud Cyberinfrastructure Security

AI-Assisted Malware Detection and Classification in Cloud CI

AI4SecureCI is a research and education website for AI/ML-assisted malware detection, online behavioral monitoring, and malware-family classification in cloud cyberinfrastructure.

View Modules Project Details

Focus Areas

  • Online malware detection in cloud CI
  • Runtime behavior and API-call analysis
  • AI/ML-based detection and classification
  • LLM-assisted cybersecurity education
  • Safe, isolated lab workflows

Project Overview

This project develops educational materials and hands-on learning activities for understanding how artificial intelligence and machine learning can support malware detection and classification in cloud cyberinfrastructure. The curriculum emphasizes behavioral analysis, runtime monitoring, safe laboratory practices, and careful evaluation of machine-learning models.

The current public website is configured for the GitHub Pages route https://ai4secureci.github.io/AI4SecureCI/. All page links and asset paths use relative URLs so the site works correctly under the /AI4SecureCI/ project path.

Curriculum Direction

The project centers on two connected learning goals. First, students learn how cloud workloads can be monitored for suspicious behavior after software has entered an environment. Second, students learn how detected malware behavior can be classified into meaningful malware types or families using behavioral features, API-call patterns, and machine-learning methods.
